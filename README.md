# Geomaster
A project for a coursework during third semester in UNECON bachelor studying. Is based on Python 3.11, SQLite 3.47, Pygame 2.6.1, Numpy 2.3.5, Subprocess and works without problems only on macOS, because of some specific settings.

In Russian:
Это прототип игры, концепция которой заключается в бесконечном решении геометрических головоломок, генерирующихся в зависимости от предыдущих успехов пользователя. Игра выполнена без использования framework, но с концепцией, напоминающей структуры Spring-проектов, так как Java мне ближе, но в вузе надо было выполнять курсовую именно на python. Помимо базового функционала генерации и подбора задач, в игре доступна вкладка с базами данных, настроек и реализована система аккаунтов. Поле для рисования – это граф с 56 вершинами, который хранится в качестве списков смежности. Геометрические ищутся через DFS. Уровень пользователя рассчитается по количеству ошибок, сложности аадачи и верных решений. Подробная информация указана в отчете, прикрепленном к репозиторию.
